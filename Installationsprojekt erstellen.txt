Erstellen eines Installationsprojekts in Visual Studio
======================================================



1. Voraussetzung
----------------
	Es muss die "Microsoft Visual Studio Installer Projects extension" installiert sein
	Dazu -->  Extensions --> Manage extensions  aufrufen.

https://learn.microsoft.com/de-de/previous-versions/visualstudio/visual-studio-2010/k3bb4tfd(v=vs.100)

https://learn.microsoft.com/de-de/visualstudio/deployment/installer-projects-net-core?view=vs-2022



2. Installationsprojekt erstellen
---------------------------------
--> Menü Datei --> auf Hinzufügen --> Neues Projekt.
--> im Eingabefeld "Suchen"  nach "Setup" suchen
--> "Setup-Projekt" auswählen:

	Maske "Neues Projekt konfigurieren:
	- Projektname: "EosMonitorSetup"
	- Ort:  "C:\Users\Herbert\source\repos\EosMonitor\" auswählen
	- Projektmappe:  "Hinzufügen"
	--> Erstelen

3. Das Fenster "File System (...MonitorSetup)"  wird geöffnet

4. Im Projektmappen-Explorer:
	- Das Projekt "EosMonitorSetup" auswählen
	- ProductName:  "EosMonitor"
	- Rechtsklick auf "Application Folder"  --> Rechtsklick:  "Add"  --> Projektausgabe:
	- Projekt "EosMonitor"
    - Für .Net-Projekte:  "Veröffentlichungselemente" auswählen  (sonst: "Primäre Ausgabe")
	- Konfiguration : "Aktiv"
	
6. Ein Veröffentlichungsprofil für das Anwendungsprojekt "EosMonitor" erstellen:
	"Veröffentlichen" aus dem Kontextmenü von EosMonitor im Projektmappen-Explorer 
	auswählen und die Anwendung in einem Ordner bereitstellen.	
	
7. Detected Dependencies sollte enthalten:
   Microsoft ".Net Framework"  und "WinRT Runtime.dll" 
	
8. Weitere Elemente zum Application folder Hinzufügen:
	- EDSDK.dll
	- Edsimage.dll
	- EosMonitor.cfg
	- EosMonitor-Anleitung.pdf
	- EosMonitorIcon.ico
	- PDF_Icon.ico
	- Rechtsclick auf Veröffentlichungselemente ... --> Create Shortcut to EosMonitor.exe
      --> Den Shorcut nach "Users Desktop ziehen	
	- Rechtsclick auf EosMonitor-Anleitung.pdf --> Create Shortcut to EosMonitor-Anleitung.pdf
      --> Den Shorcut nach "Users Desktop ziehen	
	
9. Default-Ausgabe-Verzeichnisse umstellen:  --> Projekt --> Eigenschaften

   - Default:   	C:\Users\Herbert\source\repos\Setup1\Setup1\Release
					C:\Users\Herbert\source\repos\Setup1\Setup1\Debug
					
   - ändern in:		C:\_work\EosMonitor_2025-05-11\EosMonitor_Setup\Release
					C:\_work\EosMonitor_2025-05-11\EosMonitor_Setup\Debug
