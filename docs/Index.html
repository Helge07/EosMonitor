DefocusStacker

 

Defocus Stacking ist ein Verfahren, das die Hintergrund-Unschärfe von Bildern mit Hilfe eines Stapels von Bildern verstärkt. Der Stapel besteht aus einer scharfen Aufnahme und darunter weiteren Aufnahmen, bei denen die Schärfeebene zur Kamera hin verschoben wurde, so dass die Hintergrund-Unschärfe zunimmt. DefocusStacker ist eine Implementierung des Verfahrens, das die Wirksamkeit der Methode demonstriert. Es gibt zwei Versionen:

  • eine Implemtierung für Windows,

  • eine Implemtierung als Photoshop Aktion
