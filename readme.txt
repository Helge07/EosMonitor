EosMonitor is a tethering app for Canon DSLR- and DSLM-cameras

EosMonitor is a Windos WPF application, written in C# based on .Net 9.0. and the Canon development kit EDSDK 13.18.40.  

Cameras are connected to the host computer via USB. EosMonitor sets the parameters for taking pictures, can display a liveview image and download pictures to the host. A special feature of EosMonitor is the ability to take focus stacking and defocus stacking image series.
