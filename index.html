EosMonitor

 

EosMonitor ist eine Thether-Anwendung für DSLR- und DSLM-Kameras von Canon, die über USB-Kabel an einen Windows-Rechner angeschlossen sind.

  • EosMonitor stellt die Aufnahme-Paramer ein,

  • kann das LiveView-Bild anzeigen und

  • aufgenommene Bilder auf den Host übertragen.

Ein besonderes Merkmal von EosMonitor ist die Aufnahme von Focus- und Defocus-Stacks.
