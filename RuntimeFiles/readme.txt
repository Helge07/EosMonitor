
If you compile the EosMonitor project with Visual Studio 2022 the binaries 
(EosMonitor.exe an others) are put into the directory 
EosMonitor\bin\Debug\net9.0-windows10.0.22621.0
resp.
EosMonitor\bin\Release\net9.0-windows10.0.22621.0

Before running EosMonitor.exe add the following three files 
to the above directories:
1. EDSDK library files from the Canon development kit EDSDK v13.18.40
- EDSDK.dll
- EdsImage.dll
2. The EosMonitor configuration file 
EosMonitor.cfg
